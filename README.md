# Automate Operation

Salesforceを対象に、操作を自動化

## 概要 - Description
ブラウザを起動せずにローカルで実行できるSOQLのクライアントツール。

## 利用方法 - Usage
### インストール - Install

1. Node.jsをインストールします。
1. 「npm install」を実行します。
1. .envファイルを作成します。


### 実行方法 - Run
terminalで「node automation_tools.js」を実行します。

### 操作方法 - Operation
HELPを参照してください。

## ヒント - Hints
### 環境設定 - Env
.envファイルは下記の項目を設定します。
SFDC_ID={ユーザID}
SFDC_PW={パスワード}
SHEET_NAME=Sheet1
